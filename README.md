
# HeartHeal - Professional Flutter App

This project contains a professional-looking Flutter UI for a breakup tips app.

## Features
- Modern gradient hero section
- 3 featured photo cards
- 5 free breakup tips
- Premium membership card (₹1000)
- Attractive Material 3 design
- Ready for GitHub upload

## Build APK
1. Install Flutter
2. Open this project
3. Run:
   flutter pub get
   flutter build apk --release

The APK will be created in:
build/app/outputs/flutter-apk/app-release.apk
